# Orphaninstance Tap

## Available Software

| Package | Type | Description |
| :--- | :--- | :--- |
| `orphaninstance-kitty` | Cask | GPU-based terminal emulator for Linux (includes icon and terminfo) |

## How do I install these formulae?

`brew install orphaninstance/tap/<formula>`

Or `brew tap orphaninstance/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "orphaninstance/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
